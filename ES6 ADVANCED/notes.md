# Day 02 – ES6 Advanced

## Topics Covered
- Destructuring
- Spread Operator
- Rest Operator
- Array Methods

## Use Case
Efficient data handling and transformation in modern JavaScript applications.

## Key Concepts

### Destructuring
- Extract values from arrays and objects easily
- Improves readability and reduces boilerplate code

### Spread Operator (...)
- Copy arrays and objects
- Merge multiple arrays or objects
- Avoid mutation

### Rest Operator (...)
- Collect multiple values into a single variable
- Useful in function parameters

### Array Methods
- map() → transform data
- filter() → select data
- reduce() → aggregate data
- forEach() → iterate
