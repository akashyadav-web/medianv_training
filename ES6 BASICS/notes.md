## Key Concepts

### let
- Block-scoped variable
- Can be reassigned
- Avoids issues of `var`

### const
- Block-scoped constant
- Cannot be reassigned
- Used for fixed values and references

### Arrow Functions
- Shorter function syntax
- Lexical `this`
- Commonly used in callbacks and array methods

### Template Literals
- Use backticks (`)
- Supports string interpolation
- Allows multi-line strings
